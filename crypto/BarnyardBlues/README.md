# Barnyard Blues

## Category
Crypto

## Difficulty
Easy

## Description
Old Farmer Jenkins hasn’t been right since the tractor broke down. Rumor is he’s been hiding his secret hog feed formula using strange symbols scrawled all over the barn walls. The local kids say it looks like “chicken scratches in boxes.” You manage to snag a photo before the sheriff shoos you off. Can you crack the farmer’s code?

## Tools
* Any PigPen cipher solver or manual substitution
* Online resources like dCode PigPen Cipher Tool

## Artifacts
* Image containing the cipher symbols
