# Capture the Bee

## Category: Rev

Some bees don’t buzz, they’re quiet as night, Waiting for the rhythm that feels just right.<br>
When the call arrives, the hive will reveal, The bee  the honey is the prize you can steal.

## Difficulty: Hard

* Players should be familiar with the eBPF and Linux Kernel internals.
* Players should be able to perfrom binary analysis and patching.

## Time Spent

* Few hours depending on the expertise of the player.

## Tools

* A disassembler (llvm-objdump, IDA, Ghidra).
* Hex editor.
* UPX (Ultimate Packer for eXecutables).

## Infrastructure:

* An EC2 instance for each player.
* Provide `ctb.o` after submission of flag 1.

## Artifacts

* Only `obfuscated/ctb` and `obfuscated/ctb.o` object file should be shared with participants.
